LabArchives API Client
======================

``labapi`` is a Python client for the LabArchives API. It provides an 
object-oriented interface for managing notebooks, directories, pages, and entries.

The library handles authentication, XML parsing, and tree traversal to 
support integration with automated workflows and scripts.
